#  Decoding Digital Behaviour: A Deep Dive into Mobile Usage using Tableau

## Project Overview
This project focuses on analyzing and visualizing mobile usage behavior across different cities and demographic groups using Tableau. By converting raw smartphone usage data into interactive visualizations, the project uncovers meaningful patterns related to screen time, app preferences, gender differences, and geographic trends. The aim is to promote digital wellbeing and enable data-driven insights through effective information visualization.

---

## Objectives
- Analyze city-wise mobile usage patterns
- Study demographic trends based on age and gender
- Compare app usage categories (Social Media, Gaming, Productivity)
- Identify relationships between screen time and number of apps used
- Develop an interactive Tableau dashboard for exploratory analysis

---

## Dataset Information
- **Dataset Name:** Smartphone Usage and Behavioral Dataset
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/bhadramohit/smartphone-usage-and-behavioral-dataset
- **Records:** 1,000 users
- **Format:** CSV (Tabular)

### Attributes
**Categorical**
- Gender
- Location (City)

**Quantitative**
- Age
- Total App Usage Hours
- Daily Screen Time Hours
- Number of Apps Used
- Social Media Usage Hours
- Gaming App Usage Hours
- Productivity App Usage Hours

## Visualizations Created
1. Average Age Distribution by City (Bar Chart)
2. Gaming App Usage vs Number of Apps Used (Scatter Plot with Clustering)
3. City-wise Social Media Usage (Tree Map)
4. Gender Distribution and Screen Time by City (Geographical Pie Map)
5. Screen Time Distribution by Gender (Stacked Bar Chart)
6. App Category Usage by City (Heatmap)
7. Productivity App Usage by City (Bubble Chart)

---

## Dashboard Description
The interactive dashboard combines all visualizations into a unified view. It includes summary statistics such as average daily screen time, average number of apps used, and top-performing cities. Users can explore behavioral patterns across cities, genders, and app categories using interactive filters and linked charts.
